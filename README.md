# Outfit Recommendation System [FIT_CHECK] (DSA Project)

A C++ project that applies core Data Structures and Algorithms concepts to solve a real-world problem: outfit selection based on weather and preferences.

## Features
- Weather-based outfit recommendation
- Mix-and-match clothing logic
- Sorting and searching outfits by season
- Undo functionality using stack
- Outfit history using linked list
- File handling for data persistence

## Concepts Used
- Arrays & Structs
- Sorting & Searching
- Stack
- Linked List
- Rule-based algorithms
- File Handling (Text)
